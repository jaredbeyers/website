Automated Plant Watering System

Smart irrigation system with environmental monitoring

Project Overview

This automated plant watering system uses a Raspberry Pi to monitor soil moisture levels and automatically water plants when needed. The system includes environmental sensors for temperature and humidity monitoring, with data logging capabilities and mobile alerts.

Raspberry Pi 4 Python GPIO Control Soil Moisture Sensors DHT22 Sensor Relay Module SQLite Database Web Interface

Key Features

  • Automatic soil moisture monitoring with multiple sensors
  • Programmable watering schedules and thresholds
  • Real-time temperature and humidity monitoring
  • Data logging and historical analysis
  • Mobile notifications for system status
  • Web-based control interface
  • Water level monitoring and alerts
  • Energy-efficient operation with sleep modes

System Architecture

The system consists of a Raspberry Pi 4 as the main controller, connected to various sensors and actuators through GPIO pins. A relay module controls the water pump, while soil moisture sensors provide feedback for intelligent watering decisions.

Main Control Loop

import RPi.GPIO as GPIO import time import sqlite3 from datetime import datetime class PlantWateringSystem: def __init__(self): self.moisture_sensor_pin = 17 self.pump_relay_pin = 18 self.dht_pin = 4 GPIO.setmode(GPIO.BCM) GPIO.setup(self.pump_relay_pin, GPIO.OUT) GPIO.setup(self.moisture_sensor_pin, GPIO.IN) def read_moisture(self): return GPIO.input(self.moisture_sensor_pin) def water_plants(self, duration=5): GPIO.output(self.pump_relay_pin, GPIO.HIGH) time.sleep(duration) GPIO.output(self.pump_relay_pin, GPIO.LOW) def run(self): while True: moisture = self.read_moisture() if moisture == 0: # Dry soil self.water_plants() self.log_event("Plants watered") time.sleep(300) # Check every 5 minutes
