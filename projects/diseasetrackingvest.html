Wearable Disease Tracking Vest

A Raspberry Pi-enabled wearable device with multi-sensor technology to accurately track daily contact interactions.

About the Project

The Wearable Disease Tracking Vest is an innovative health monitoring system designed to accurately track and analyze human contact patterns throughout the day. The system consists of four distributed subsystems that collect sensor data and transmit it to a central processing unit for comprehensive analysis.

Each subsystem is equipped with a Raspberry Pi Zero 2, MLX90640 thermal camera, VL53L7CX time-of-flight distance sensor, and an Arducam day/night camera. These subsystems continuously monitor the environment and detect potential contacts within a 2-meter radius.

The central system features a Raspberry Pi 5 and a GPS module for accurate location tracking. An OpenCV model running on the Pi 5 automatically processes the collected data to track the number of interactions, duration of each interaction, location of each interaction, and the closest distance reached per interaction.

  • Four distributed subsystems, each with Raspberry Pi Zero 2 for local data collection
  • MLX90640 thermal camera for temperature-based detection
  • VL53L7CX time-of-flight distance sensor for precise proximity measurement
  • Arducam day/night camera for visual detection and verification
  • Central Raspberry Pi 5 processing unit with GPS module for location tracking
  • OpenCV-based computer vision model for automated interaction analysis
  • Automatic tracking of interaction count, duration, location, and closest distance
  • 2-meter contact detection radius for accurate proximity monitoring
