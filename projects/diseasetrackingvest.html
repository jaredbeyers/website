Wearable Disease Tracking Vest

A Raspberry Pi-enabled wearable device with multi-sensor technology to accurately track daily contact interactions.

About the Project

The Wearable Disease Tracking Vest is an innovative health monitoring system designed to track and analyze human contact patterns throughout the day. Built with a Raspberry Pi as the central processing unit, this wearable device integrates multiple sensors to provide accurate, real-time data on interpersonal interactions and proximity events.

  • Raspberry Pi-based processing unit for real-time data collection and analysis
  • Multi-sensor array including proximity sensors, accelerometers, and environmental sensors
  • Accurate contact detection and counting algorithms
  • Daily interaction tracking with timestamp logging
  • Wearable design optimized for comfort and functionality
  • Data visualization and reporting capabilities
